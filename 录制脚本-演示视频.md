# 🎬 评委演示视频录制脚本（3 分钟版）

> 录制要求：横屏 16:9，1920×1080，F11 全屏，系统内录音频，无控制台/标签页干扰

---

## 录制前检查

- [ ] 浏览器全屏（F11）
- [ ] DevTools 关闭
- [ ] 系统音频内录开启
- [ ] 有线网络（避免加载延迟）
- [ ] 浏览器隐身模式（干净状态首次体验）
- [ ] Demo 链接正常：`http://localhost:5173` 或线上地址

---

## 时间线脚本

### [0:00-0:05] ① 首屏第一印象 — 暗黑水墨

```
画面：浏览器全屏打开 Demo
操作：无（自动加载）
预期：
  - 暗黑水墨底色，非白屏
  - "山海寻迹"毛笔字标题带赤红发光
  - "考山海之经，寻上古神兽踪迹"引导文案
  - "开始探索山海" + "寻我守护神" 双金色按钮
  - 右侧 AI 对话预览（HeroChatPreview）
配文（后期字幕可选）：
  不用加字幕，让画面自己说话
```

### [0:05-0:15] ② 入口感知 — 进入地图

```
画面：鼠标悬停在"开始探索山海"按钮上
操作：点击"开始探索山海"
预期：
  - 平滑滚动到地图区域
  - "仙家舆图"标题出现，各省份颜色不同（卷册设色）
  - 地图下方有图例 + 引导文字
配文：
  "34 省交互地图，按《山海经》原书五卷设色"
```

### [0:15-0:55] ③ 遭遇动画 — SSR 紫电雷劫

```
画面：鼠标移动到陕西省
操作：点击西安/陕西省
预期：
  - 迷雾散开效果
  - 7 阶段遭遇动画开始
  - 暗幕→水墨扩散→[古文浮现]→[法阵旋转]→[神兽显现]→[金光炸裂]→[仙名题刻]
  - SSR 级有紫电雷劫特效（屏幕上方紫色闪电）
  - WebAudio 低音+和声+仙铃
  - 全程流畅无卡顿
操作：不按跳过，看完完整动画（SSR ~4 秒）
配文：
  "7 阶段遭遇动画——SSR 神兽有紫电雷劫！"
```

### [0:55-1:20] ④ 方言朗读 — 听声按钮

```
画面：动画结束后自动弹出神兽卡片（毕方为例）
操作：
  1. 看到"毕方"标题、"✦✦✦"SSR 标识
  2. 点击"听声"按钮（或以 AudioPlayer 组件播放）
预期：
  - 方言音频开始播放
  - 毕方用陕西话朗读"有鸟焉，其状如鹤，一足，赤文青质而白喙…"
  - 声音自然，有古朴庄重感
  - 播放期间有视觉反馈（进度条/波纹）
操作：
  保持麦克风安静，让观众听清方言朗读
  
⚠️ 这是评委 "哇时刻"——确保音频不卡顿、声音清晰
配文：
  "毕方用陕西话朗读《山海经》原文"
```

### [1:20-1:50] ⑤ AI 对话 — 打字机效果 + 羁绊记忆

```
画面：神兽详情页
操作：
  1. 点击"对话"按钮（或自动展开 CreatureChat）
  2. 输入 "毕方，你当年在哪个山上？"
  3. 回车发送
预期：
  - AI 回复 1-2 秒内开始
  - 打字机效果逐字显示古文式回复
  - 回复内容符合毕方角色设定
  - 回复带有明确的当前时间概念
  示例回答："吾曾栖于章莪之山，今陕西境也。彼时满山赤石…"
操作：
  输入第二个问题 "那你怕什么？"
预期：
  - AI 记住上文（羁绊记忆注入）
  - 回复节奏自然
  - 用户能感受到"这不是预设剧本"
配文：
  "每只神兽都是独立的 AI Agent——有性格、有记忆"
```

### [1:50-2:20] ⑥ 功能巡览 — 图鉴 + 守护神 + 分享

```
画面：快速展示其他核心功能
操作：
  1. 返回首页 → 点击"寻我守护神"
  2. 展示定位/手动选择省份 → 守护神卡片弹出
  3. 点"生成分享海报" → Canvas 渲染 → 展示下载的 PNG
  4. 返回 → 进入图鉴（BestiaryView）
  5. 展示五卷分类、搜索、渲染效果
  6. 浏览一只 3D 模型神兽 → 旋转展示
预期：
  - 守护神定位正常
  - 分享图生成清晰（750×1334）
  - 图鉴列表正常
  - 3D 模型 Bloom 泛光 + 粒子 + 法阵效果
配文：
  "图鉴收藏 · 家乡守护神 · 分享海报 · 3D 神兽预览"
```

### [2:20-2:50] ⑦ 收尾 — AI 可见度 + 完整度

```
画面：静态展示 + 快进式巡览
操作：
  1. 展示首页上方统计数字（104 神兽 / 34 省份）
  2. 快速滚动神兽详情页展示 原文/译文/考据/关系网络
  3. 展示节气 Banner + 任务系统
  4. 展示羁绊面板（BondPanel）
  5. 快速浏览山海卷轴（ShanhaiScroll）水平滚动
预期：
  - 各项系统正常显示
配文（最后画面停留 5 秒）：
  "104 只神兽 · 34 种方言 · 一个 App"
  "山海寻迹——让《山海经》开口说话"
```

### [2:50-3:00] ⑧ 结束

```
画面：回到首页，或空白暗幕
配文：
  Demo 链接 + 二维码（可选）
```

## 视频 ①：你的家乡守护神是谁？

**目标**：利用地域认同 + 好奇心驱动分享

**时长**：15-20 秒

**录制操作**：
```
[0:00-0:03] 打开 App 首页（暗黑水墨 + "山海寻迹"标题）
            → 标题：你的家乡有没有一只你不知道的守护神？

[0:03-0:08] 点击"寻我守护神"按钮
            → 定位动画 / 手动选择省份

[0:08-0:14] 守护神卡片弹出（神兽图 + 名称 + SSR 标识）
            → 放大展示守护神卡片
            → 标题：这就是你的家乡守护神！

[0:14-0:18] 展示"生成分享海报"按钮
            → 分享卡生成画面
            → 字幕：扫码测测你的守护神
            → 右下角放 Demo 链接二维码（后期叠加）
```

**话题标签**：`#TRAE大赛 #山海经 #守护神 #AI`

**背景音乐**：古风轻音乐（如《无羁》纯音乐版）

## 视频 ②：九尾狐用四川话念《山海经》

**目标**：AI 神奇感 + 反差萌

**时长**：20-25 秒

**录制操作**：
```
[0:00-0:03] 九尾狐神兽卡片或详情页
            → 画面定格在九尾狐画像
            → 标题：听九尾狐用四川话念《山海经》
            → 省份标签：四川

[0:03-0:16] 点击"听声"按钮 → 方言朗读响起
            → 音频波形可视化（如果有）
            → 字幕同步显示原文：
            "青丘之山有兽焉，
             其状如狐而九尾，
             其音如婴儿，
             能食人，食者不蛊。"

[0:16-0:22] 展示九尾狐所在省份地图标注
            → 你的家乡有没有这样的神兽？
            → 左下角 Demo 链接（后期叠加）
```

**话题标签**：`#TRAE大赛 #方言 #山海经 #AI语音 #四川话`

**关键**：**这是全场最核心的传播素材**——方言朗读+古籍的组合有天然病毒系数

## 视频 ③：SSR 神兽现身！雷劫特效

**目标**：视觉冲击 + 炫耀

**时长**：15-20 秒

**录制操作**：
```
[0:00-0:02] 画面：中国地图，点击某个省份
            → 标题：SSR 神兽现身！

[0:02-0:12] 7 阶段遭遇动画全程（可加速到 1.2x）
            → 暗幕→水墨扩散→古文打字→法阵旋转
            → SSR 紫电雷劫特效！（全屏闪电）
            → 神兽卡片弹出 + ✦✦✦天品标识
            → 标题：⚡ 紫电雷劫 ⚡

[0:12-0:18] 放慢展示神兽卡片完整画面
            → 神兽名 + 渡劫成功提示
            → 右下角：你也来试试？
            → Demo 链接二维码（后期叠加）
```

**话题标签**：`#TRAE大赛 #SSR #特效 #山海经 #AI`

**备选**：可录制不同的 SSR 神兽遭遇（毕方、应龙、麒麟等），做多条素材
